Fox & Rabbit

Java · Data Structures · Turn Engine

Original version of the chase game on a 6×6 ring board, Advanced Programming Class Project. The Rabbit starts in the center and runs for rim carrots; the Fox starts on the outer ring, moves 3 steps, and hunts by scent. The Rabbit moves 2 steps (or 3 after a failed Fox pounce). Carrots appear on the rim, and when one is taken its two neighbors are locked out.

This project was completed for an Advanced Programming course; code access is not public.
